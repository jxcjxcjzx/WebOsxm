司机胆小是对法律的敬畏
 
上一篇 / 下一篇  2013-01-07 09:05:20 

查看( 70 ) / 评论( 0 ) / 评分( 0 / 0 ) 



山东烟台市莱山区的刘女士驾龄快一年了,算是新手到老手的过渡,不过最严交规的实施,让她似乎又回到了刚开车的那段时间,每次开车都很紧张。刘女士说,刚买车那段时间她好长时间都没动车,就怕出门出事故,走到路口也是特别紧张,生怕走错了。这种情况一直到半年后才有所好转。可是没想到车刚开溜了,最严交规却来了。（《齐鲁晚报》1月5日）

新交规规定，闯黄灯一次罚6分，这让那些驾驶员很不适应，开车经过十字路口时如履薄冰。面对新交规司机胆小了，可以说这是好事，既是对法律的敬畏，也是对安全的谨慎，如果每一个司机都能像新手那样小心翼翼地驾驶，相信会避免很多交通事故。如果每一个人都能严格遵守法律法规规定，相信很多事故不会发生。

正是因为司机技术熟练了，胆子大了，也就忽视了安全，于是频频闯黄灯。因为闯黄灯现象普遍，诱发了很多交通事故，所以，公安部出台了新交规，规定闯黄灯一次记6分，这被称为是史上最严的交规。而闯黄灯扣6分，也引起了社会的争议。因为闯了黄灯，很多驾驶人从自己的利益出发，认为新交规“违背了牛顿第一定律”，还有人认为新交规是“忽视民意闭门立法”。只是，面对闯黄灯，那些闯黄灯的驾驶人在经过十字路口的时候减速了没有？保持安全车距没有？为什么别人没有闯黄灯呢？

新交规实施后，虽然也有闯黄灯现象，也发生了为避免闯黄灯而诱发的追尾事故，但与新交规实施前，交通事故是大为减少的。据北京、天津、南京、杭州、济南等城市统计，新交规实施首日接报的交通事故数量分别比前日下降了9.3%、23%、26.7%、18.4%、29.8%，未发生因违反交通信号指示造成的恶性交通事故。由此可见，新交规“闯黄灯罚6分”，还是立竿见影的。

任何一部法律，都要经过实践的检验，为了更合理，法律也会进行修改。因为闯黄灯现象的普遍，以及导致的交通事故频发的事实，规定“闯黄灯罚6分”是必要的。但是，因为相关配套事实没有跟上，“闯黄灯罚6分”被认为是不合理，甚至不合法，遭到了驾驶人的强烈反对。其实，看一部法律是否合法，看下位法与上位法是否冲突，如果并不冲突，就不能认为是违法。，“闯黄灯”条款与《道路交通安全法》和《道路交通安全法实施条例》两部上位法并不冲突。而一部法律是否公平合理，应该照顾大多数人的利益，同时也要照顾弱者的利益。比如判抢劫、盗窃违法，就是照顾了大多数人和弱者的利益。

新交规遭到质疑，是因为有的人很难改变几年甚至十几年的驾驶习惯，因为自己受到了约束，所以就将怨气发泄到新交规的头上。诚然，新交规在执行上，有所欠缺，但不代表新交规就是“闭门立法”。面对“闯黄灯”相关条款引起的争议，公安部将出台相关实施细则，对“闯黄灯”行为进行细化。到时候，新交规的争议将会因为细化而变得合理，争议也将随之平息。

“闯黄灯扣6分”的目的，是培养驾驶人在路口减速慢行的习惯，也是培养驾驶人遵守交通法规的意识，如果每一个公民都能对法律保持必要的敬畏，小心驾驶，那么，还会去闯黄灯吗？交通事故还会频频发生吗？
