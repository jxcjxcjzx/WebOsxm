Tizen

展开编辑本段简介泰泽是基于Linux的开源软件平台，可运行在智能手机、平板、上网本、车载信息系统和智能电视上。它将由一个技术指导委员会管理，成员包括了英特尔和三星。泰泽的应用程序接口是基于HTML5和其它web开放标准。除了HTML5应用程序外，泰泽的SDK将包括原生开发工具包。从核心系统到核心应 用，泰泽的整个软件堆栈都会对外开放。泰泽的首个版本和SDK在2012年第一季度发布。[1]
编辑本段发展历史产生背景
  
Tizen系统是英特尔MeeGo系统与三星LiMo系统的混合体。 随着惠普放弃开发webOS智能手机和平板电脑，RIM的Playbook艰难求生，诺基亚剧变，运营商们希望能出现一个可替代的新操作系统，没人希望看到Android和iOS双寡头垄断的局面。 同时由于英特尔、三星各自在MeeGo及Bada发展遭遇困难，双方便携手全新平台Tizen系统，这样可以降低开发资源与人力，并强化技术及应用程序商店功能，包括宏碁、华硕均将加入Tizen平台开发计划，宏达电也正评估中，Tizen平台是否有机会形成 OS 平台第4势力，与Android、iOS、Windows相较劲，备受业界关注。 NB业者表示，由于英特尔、三星体认到自家MeeGo与Bada OS难成主流，恐将影响各自在市场地位，尤其实力最为接近苹果(Apple)的三星，面对Google收购摩托罗拉移动后续可能产生Android授权疑虑，三星不愿主控权在Google或微软(Microsoft)手上，因此，希望藉由与英特尔合作，让自家Bada平台整合重生。 至于英特尔MeeGo同样难获手机、NB业者青睐，NB业者指出，英特尔砸重金投入MeeGo开发，主要是希望能在软件系统平台地位已凌驾硬件的市况中，力守市场领导地位，但由于技术瓶颈、兼容性及应用程序等劣势，使得市场接受度偏低，难在iOS、Android及Windows厮杀战场中占有一席之地。 在此情况下，英特尔遂思考与其他 OS平台合作的可能性，与三星携手推出Tizen是不得不的决定，虽然英特尔不再具有完全主控权，但与三星合作至少可取得在终端产品应用契机，三星势将采用英特尔下一代针对平板计算机、智能型手机而生的Medfield处理器平台，搭配Tizen OS推出相关产品，英特尔藉此抢进最为弱势的平板计算机与智能型手机战场。[2]
诞生
北京时间2011年 9 月 28 日消息，据国外媒体报道，两大Linux联盟 LiMo Foundation和 Linux Foundation周三宣布将整合资源优势，将携手英特尔、华为和三星电子，共同开发针对手机和其它设备的新操作系统 Tizen。
不过行业分析师指出，Tizen 操作系统将很难吸引到更多的开发者和制造商，原因是移动操作系统市场当前已被苹果IOS和谷歌的 Android所统治，而且市场中还有 10 余款不同的移动操作系统。在此之前，诺基亚和惠普都已放弃了自己的移动操作系统。
市场调研公司 Strategy Analytics分析师尼尔·莫斯顿（Neil Mawston）表示，“对 LiMo Foundation 和 Linux Foundation 而言，最大的希望莫过于大型的移动运营商担心 Android 会统治移动操作系统市场，决定采用其它的操作系统来限制谷歌在移动操作系统市场的影响力。”
LiMo Foundation 和 Linux Foundation 称，Tizen 是开源的、标准化的操作系统。该操作系统可支持智能手机、平板电脑、智能电视、上网本以及车内信息娱乐系统等多种设备。据悉，Tizen 操作系统最初版本将于 2012 年第一季度发布，从而保证其支持的相关产品能于 2012 年中期上市。
作为全球最大的芯片制造商，英特尔还拥有自己的操作系统 MeeGo，而全球第一大手机制造商三星电子，一直是 LiMo Foundation 的主要捐助方之一。英特尔与三星电子将会领导 Tizen 的技术指导委员会。
电信产业资讯公司 CCS Insight分析师杰夫·布雷伯（Geoff Blaber）表示，“Tizen 旨在统一大量不同规格的基于 Linux 的平台。”不过该分析师同时表示，“尽管市场中有开发一款新操作系统来替代 Android 的意愿，但历史经验告诉我们，这种意愿并不意味着成功。”
三星电子目前是使用谷歌 Android 操作系统的主要电子产品制造商，但是在谷歌斥资 125 亿美元收购摩托罗拉移动之后，一些 Android 手机制造商已开始考虑使用其它的操作系统。三星电子发言人表示，三星电子是 Linux 的核心合作伙伴，公司此举乃是与提倡支持多种操作平台的战略规划相一致。
本月初，英特尔宣布与谷歌建立伙伴关系，共同开发兼容 Android 操作系统的英特尔凌动处理器，预计第一款采用凌动处理器的 Android 手机将在明年上半年上市。
Tizen 将为应用程序开发者们提供一个稳定灵活的基于HTML5与WAC（Wholesale Applications Community ）的开发环境。由于HTML5的稳定的性能与灵活的跨平台特性，它将迅速地发展为移动应用与服务的优先开发环境。Tizen SDK（Software Development Kit）软件开发工具包与API（Application Programming Interface）即应用程序编程接口允许开发者们使用HTML5及相关网络技术编写适用于包括智能手机，平板电脑，上网本，车载信息娱乐设备，智能电视等在内的多种设备的应用程序。
编辑本段版本发布Tizen 1.0
Tizen 1.0 SDK及源代码已于4月30日发布[3]。泰泽是基于Linux的开源软件平台，可运行在智能手机、平板、上网本、车载信息系统和智能电视上。它将由一个技术指导委员会管理，成员包括了英特尔和三 星。泰泽的应用程序接口是基于HTML5和其它web开放标准。除了HTML5应用程序外，泰泽的SDK将包括原生开发工具包。 
该版本包含一些新特性和改进，SDK 的更新包括：
仿真器: 一个全新基于浏览器的工具，你可直接在Web 上运行和调试 Tizen API，并在不同的设备上模拟执行
IDE: 提供更灵活的模板和调试工具
模拟器: 透过 Intel 的硬件加速管理器显著的改善了模拟器的性能
平台源码更新包括：
Web: 支持 W3C/HTML5 规范的附加特性
Location: 支持 POI (Point of Interest) and route search
Connectivity: Wi-Fi Direct key features added
Tizen 2.0 Alpha
Tizen 2.0 alpha 增加了更多的特性和工具，以及其他提升，包括：[4]
增强的Web 框架，提供了更好的HTML5/W3C API 支持和更多的Tizen 设备API
多进程基于Webkit2的Web Runtime ，以提供更安全和稳定的Web应用
支持最新的HTML5特性，如video subtitles and captions, battery status API, screen orientation API, <keygen> and <details>，以及更多
新的Tizen 设备APIs：文件传输，通知和电源控制
高级Web应用开发IDE和SDK
基于快照的网络安装管理器支持
增强的OpenGL ES支持
新的帮助开发基于OBS平台的Platform SDK
编辑本段外界评价AMD内容和程序支持负责人Neal Robison日前在访谈当中谈到了他对目前移动设备和PC操作系统的看法。
Neal Robison表示，现在投资Intel MeeGo或者惠普的WebOS毫无意义。现在各种设备操作系统划分已经非常成熟，其中包括Google安卓、苹果iOS和微软的Windows。
Neal Robison认为，这三大操作系统已经可以满足所有和各类用户需求。如果投资这三大操作系统以外的它操作系统，投资不会有产出，其中包括英特尔和三星电子合作研发的Tizen操作系统。
在英特尔MeeGo操作系统失败之后，英特尔又和韩国三星电子联手准备推出跨平台的Tizen操作系统。Tizen同时适用于智能手机、平板和上网本等设备。据称，Tizen结合了Limo和Linux操作系统的长处，并且内建基于HTML5和WAC的标准webb开发环境，可以跨平台为设备研发应用程序。
Neal Robison表示，AMD会采取和英特尔不同的道路，专注于为已经取得市场成功的操作系统进行产品研发。
三星携50万美元加入Linux基金会,欲发展Tizen!　三星将宣布加入Linux基金会，其加盟所携的50万美元也使其一跃成为该联盟的白金会员，并在董事会拥有一席之地。
业内人士认为，三星进入Linux基金会董事会将有利于其更好掌握Linux平台即将推出的相关产品，其中包括Tizen移动操作系统。
由于目前移动操作系统格局中，iOS和Android占据主导地位，而Windows Phone也蓄势待发，因此很少有人注意Tizen的存在。不过，三星却是Tizen的有力支持者。
Linux基金会发言人说，三星在今年第一季度超越诺基亚成为世界第一大手机制造商，如今加入Linux基金会更加表明三星准备通过Android和Tizen平台来挑战苹果在该领域的地位。
针对目前iOS在与Android的竞争日渐占据上风，三星并不想坐以待毙，Tizen就是很不错的后备计划。
Linux基金会的白金会员需要大约50万美元，目前只有IBM、甲骨文、英特尔、富士通、NEC、高通创新中心(Qualcomm Innovation Center)以及三星达到了这一级别。
