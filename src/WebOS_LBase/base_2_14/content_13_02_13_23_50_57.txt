挑战ETS之新托福阅读解题方法介绍
寄托天下	2013-2-4	我要评论	浏览606次
本帖主要介绍解题方法，各题型的TPO例题讲解可以参看这里。

托福阅读每篇各一共14题，细分的话题型共十种两大类，详细的分类和官方建议可以看这里。在详细介绍这些题型之前，想告诉大家的是，本文的目的不是教大家多么神奇的选对答案的考试技巧，而是试着结合TPO阅读题目的实例，给大家提一下在做题时候容易忽视的小细节。读完下面的内容你就会知道，做好TPO阅读需要的能力，就是单个段落的理解和段落与段落之间逻辑关系的把握。

IBT阅读题型按照考察目的分为十种两大类：
A大类、考察基础信息理解能力及推论能力（basic information and inferencing questions）

题型1：事实细节题，3-6每篇（Factual Information Questions）    例题入口

这类题主要考察明确出现在原文中的事实细节，可能是事实，细节，定义等。我们需要认真读懂一段文字内容，剔除错误选项。
针对这种题型，提几个建议给大家：
tip1：阅读选项，仔细核对原文。切忌因为某句中重复出现了较多的原文词句就选择，读懂选项意思，符合原文内容才是做对这题的关键。
tip2：善用排除法。相比正面理解，辨析原文与选项的异同来说，排除法更加直接，效果立竿见影。

有些情况，读完四个选项也许只能剔除两个选项，剩余两个选项很难抉择，即使利用排除法也挑不出来问题，怎么办？
请记住，这些题目是我们迈向阅读高分的关键！
建议大家在练习时尤其注意这种题目，每题都要琢磨清楚，记住当时没有辨清的点在哪里。结合自己的做题经验，我个人比较难以分辨的是那种偷换概念的。比如编造因果关系、假造原文没有的前提条件、原因之一偷换成唯一原因等等。

提问方式 
According to the paragraph, which of the following is true of X?
The author’s description of X mentions which of the following… ?
According to the paragraph, X occurred because…
According to the paragraph, X did Y because…
According to the paragraph, why did X do Y?
The author’s description of X mentions which of the following?

题型2：反面事实细节题（Negative Factual Information Questions）0-2每篇    例题入口

这类题还有一个更为人熟知的名字，叫except题。要求我们选出不是事实或未出现在文中的信息。
Except题考查的段落范围更大，往往需要在全文范围内寻找做题依据。值得注意的一点是，正确选项可能是和原文内容相悖，同时也很可能是原文从未出现过的内容！然而，找没有的内容要比找有的内容难太多，所以建议大家尽可能使用排除法。另外，相当一部分except题考察的选项存在并列关系。简单来说，选项都是对应段落的核心内容，所以在我们阅读文章时候，留心并列关系有时候可以起到奇效。

提问方式
According to the passage, which of the following is NOT true of X?
The author’s description of X mentions all of the following EXCEPT..

题型3：推理题（Inference Questions）0-2每篇    例题入口

众所周知的infer题，主要考察我们理解被强烈暗示，却没有被明白表达的观点的能力。如果题目中明确给出了定位词，建议大家先根据题干中的定位词寻找到原句，否则需要先正确理解文章的含义。做题时尽量避免被前后冗余信息干扰，抓住作者真正要表达的核心观点。

比较常见的推理思路有一下几条
第一条是取非式，标志词有unlike, different from, in contrast to, another等等，也就是没有明说的这部分应该具有和直接描述特性的事物相反的特征。

第二条是时间式，标志词就是各种时间状语，in 1900, before 18th century等等，特定时间之前的情况可能和该时间之后的情况存在相反的情况。

第三条是程度式，标志词有than, more, less等等，相对来说比较简单。文中的两个对象存在同样的特质，但是存在数量、难易、大小等类型的程度不同，构成了程度比较推理。

第四条是类比式,比如原文叙述完一个对象所具备的一些特点，某个选项表达某相似的对象存在这一特点的意思，那么这一选项就要优先选择。另外，如果是整体的特点，一般都会被整体中的部分继承。

推理思路可以分类总结的主要是以上四种，但并不局限于此。例如一些常识，如个体数据可能大于也可能小于总体平均值，又如人类早期的以物易物贸易主要是因为不能生产需要的货物等等。还有相当部分的推理题需要我们领会文章意思，合理推断作者暗示。虽然比较笼统不好把握，但是有几个核心的原则必须把握：
1、正确的选项不应该和全文思想冲突，如果存在意思上的明显偏差，必然是错误的推理。 
2、在有更好的选项时，不优先选择较多重复原文内容的选项。 
3、推理要有依据。


提问方式
Which of the following can be inferred about X?
The author of the passage implies that X…
Which of the following can be inferred from paragraph 1 about X?

题型4：修辞意图题（Rhetorical purpose Questions）0-2每篇    例题入口

本题主要考察对文章逻辑层面的理解。为什么作者要在这里要说这个？为什么作者要这么说？一般来说，较常被问及的是文中例子的修辞意图。所以在阅读文章的时候，我们就应该有意识地留意举例论证的文字。相较于例子的具体内容，更重要的是例子前后的句子的逻辑关系。我们在意的是这个例子对文章论述起到的作用，至于说例子具体讲了什么，无关痛痒。

技巧上来说，对一些常用的单词词组要留心，比如definition, example, to illustrate, to explain, to contrast, to refute, to note, to criticize, function of 等等。还有一点，和原文意思相悖的选项，直接排除。这类题一般只要把被询问的句子往前往后至多各两句的内容读透了，把握其中的逻辑关系，问题不大。

提问方式
The author discuss X in the paragraph 2 in order to …
Why does the author mention X?
The author uses X as an example of…


题型5：词汇题（Vocabulary Questions）3-5每篇

词汇题没什么可说的，如果已经掌握了这个单词，那把近义词挑出来代入原文意思通顺就好了。说几句题外话，我觉得词汇题完全就是送分的，这一类题目不想多强调什么技巧，技巧无非就是代入法。重点是如果词汇量感觉不够，那就踏踏实实背单词，不要想着走什么不背单词的捷径，关键是花功夫努力。这里提供词汇部分资源下载入口。

提问方式
The word X in the passage is closest in meaning to
n stating X, the author means that

题型6：指代题（Reference Questions）0-2每篇    例题入口

代词主要由人称代词、指示代词等等，注意语法上代词的一个特点是就近原则，所以做题的话跟修辞意图题有点像，代词往前往后各两句句子读透了答案就有了。这类题主要考察对长句的理解，分析句子之间关系的能力。此外提两点建议

1、留意代词和所选答案之间必须单复对应，运用这点有时可以迅速排除某些选项。
2、选出答案之后代回原文重新读一下看是否通顺。

提问方式

The word X in the passage refers to…

题型7：句子简化题（Sentence simplification Questions）0-1每篇    例题入口

通俗地说，这类题目可以看做缩句。请确保简化前后两句句子的意思不能有冲突，利用这个要求，排除法往往是首选，在排除足够选项之后再深入分析剩余选项的意思与原句的关系。另外，请牢记，正确选项是不会遗漏原句中的关键信息的，所以千万不要只是因为某一选项中出现较多原句的成分而选择它。怎么样获取关键信息？那就要充分考察原句的主句和核心修饰语。
一般来说被高亮的句子一定是有容易被混淆或误读的部分；
以下几种情况要找到准确的主句（第一层次的主谓宾）：
插入语过多
从句复杂
以下几种情况要注意寻找重要从句，而非第一层次的主谓宾：
同位语从句
形式主语句后真正主句句子

提问方式
Which of the following best expresses the essential information in the highlighted sentence? Incorrect answer choices change the meaning in important ways or leave out essential information.

题型8：插入文本题（Insert Text Questions）0-1每篇    例题入口

要求把所给句子插入原文的四个空位中最合适的一个。本题主要考察理解文章逻辑结构的能力，需要注意句子和句子之间的联系。需要留心的是，四个空不一定出现在同一段中，有可能分处连续的两段内。

先寻找关键性的指代词，如代词和关联词。代词主要是人称代词和指示代词（I, this），根据代词的逻辑意思，找准指代对象.代词之前一般都会有先行词，可以查看代词与其之前的句子是否通顺，是否与需要插入的句子之间的词存在指代关系。

其次需要留心的是连接词，如on the other hand, for example, on the contrary, similarly, in contrast, further(or furthermore), therefore, in other words, as a result, finally，通过分析它们引导的内容及前后逻辑来帮助分析，看是否存在逻辑的缺失，以此插入正确的位置。

最后是一些比较特别的词，比如either，它说明该句之前应该有两个并列的内容；又如another，它说明待插入的句子和之前的句子应该是并列关系。还有等等的词，需要在做题中自己领悟。没有指代词和连接词的情况比较少，万一遇上也不用担心，仔细分析句与句之间的意思发展，还是可以选对的。

提醒大家做出选择之后把句子插入原文再读一遍，确保正确。Look at the four squares [■] that indicate where the following sentence could be added to the passage.



B大类、考察对全文结构四落的理解能力，把握要点能力，及分辨修辞功能的能力（Reading to learn Questions）

此类题目分为两种题型，但两种题型加起来每篇只会出现一题，然此类题目分值也最大

题型9：全文概要题（Prose Summany Questions）    例题入口

考察辨识论点并为各论点重要性排序的能力本题无官方Tip六选三，分值为2分：选对0个或1个无分；选对2个1分；选对3个得满分2分。不需考虑答案排列顺序。

根据做题体验，直接选出正确选项不是那么直观，而错误选项错得都比较有特点，所以建议大家多用排除法来考虑。并且，为了做题高效，尽量遇错即停，看下一个选项，以此来节省时间。
总的来说，选项的错误原因主要有如下三种：画蛇添足、黑白颠倒和本末倒置。

画蛇添足，指选项中出现了文章中根本就没有出现的内容。做这题的原则是和细节题一样的，以原文为准。一旦选项中出现没有的内容，就可以大胆抛弃。再提醒大家一次，这题不用做任何推断，所以如果选项中出现了哪怕看似非常合理的推断，也要千万慎重。

本末倒置，指选项基本上就是对原文小例子、小细节的描述或强调。本题考查的是梳理全文重点的能力，所以在这题中只突出例子和细节是不正确的。此题考察全文内容，正确选项不会出现在某一句话中，所以千万不要断章取义。

颠倒黑白，指与原文意思相反的选项。这类选项一般比较扎眼，可以第一时间发觉并排除。

当然，错误的原因主要是并不仅限于上述三种。总而言之，这题对大家理解把握全文内容和结构有很高的要求，但是只要多加练习，是肯定可以答对的。

题型10：填表题（Fill in a table Questions）    例题入口

做此题，需点击所选项，将其拖至表格中正确的栏目下。考察将文章概念化，总结归纳文章要点的能力。相较于全文概要题，这题出现概率比较低，难度也比较小。

OG中明确给出了本类题目计分的原则。分类填表题给出的表一般分为两类或者三类。供选择选项可能为5个或者7个。对于有5个正确选项得题，一共值3分，答对5个得3分，答对4个得2分，答对3个得1分，答对2个或2个以下不得分;对于有7个正确选项的题，共值4分，答对7个得4分，答对6个得3分，答对5个得2分，答对4个得1分，答对3个或3个以下不得分。

由于这样的题目形式，这类题目操作起来比较简单。一言以蔽之，就是“内容上分段，结构上分层”。表格中的两个或者三个类别，就是文章的关键词。抓住这几个关键词，理清全文的脉络，内容分段结构分层，这题就能轻松选对。
