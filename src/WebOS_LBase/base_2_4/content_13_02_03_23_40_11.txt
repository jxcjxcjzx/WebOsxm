黄祖斌：可以大胆买进下跌的茅台

上一篇 / 下一篇  2013-01-31 14:52:59
查看( 14 ) / 评论( 0 ) / 评分( 0 / 0 )
中午有网友发短信给我，认为公务消费不止占茅台消费量的几个百分点，而决策层对反腐很坚决的，一两年内茅台的销量会有负增长。
 
中国股市从来都很烂，但因为中国的投资者太妄想不劳而获，又有炒新股等恶习，于是进一步纵容了中国股市恶政横行。然而，在2011和2012年，熊市的末期，却出现数以十万亿市值的投资价值非常明显的标的。其中最主要的就是银行股，这一两年我已经写了很多文章说得很清楚了。我正是在这两年持有仓位近一半的银行股，于是躲过了熊市的风险，股市最近一两个月稍有反弹，市值就创了新高。而贵州茅台这只股票，不仅是长期投资者理想的投资标的，也是可怜又可恨的一些小散户逐步减少股市净亏损的投资标的。
 
关于茅台的投资价值，前期在我博客里有过一些详细的观点，主要投资亮点是：
1、自主定价权，就是涨价了，市场份额不下降，照样卖得掉；
2、没有长期库存积压的问题，酒的特点是存放时间越长越值钱；
3、茅台价格确实下跌了近一半，从最高的2500元/瓶左右，跌至最近的1200元/瓶左右。但茅台的出厂价去年上涨了近30%以后，也只有819元/瓶。就是说，茅台市价大幅下跌后，对茅台的利益没有影响。
4、茅台具有全国性、长期性的知名度。飞天茅台从来不做电视广告。
5、目前茅台估值非常低，就是利润零增长，也不可怕。
 
其实茅台13年销量负增长10%，但因为价格提了30%，利润也不可能负增长。
 
关于腐败和茅台的前景，我也从来不怕。我一直没有关注到底有多少茅台被消费在公务腐败上，这不是问题的关键。我的偏见是，茅台消费的增长，主要是商务宴请。茅台最近半年销量确实在下滑，这主要是经济在下滑。经济下滑这个大局，可以解释很多东西。比如去年百货商店（包括国美和苏宁）的销量，和品牌服装的销量的下滑，其实都是经济在下滑，而不是电子商务的兴起。2011和2012年的电子商务市场环境能有巨大的变化吗？不可能。
 
茅台酒的销量，失业率，和银行的不良贷款比例，其实都是经济的滞后指标，而不是先行指标或是同步指标。既然经济已经回升了，茅台的销量也一定会回升。
 
新的国家主席上台后，反腐会不会加强，我看会，但能强到哪里去，不能下绝对的定论。因为腐败是人的天性。朱元璋这个出生于农民的人反腐厉害不厉害？前任地方官腐败被杀，剥皮后，皮放在地主官署中，以警示继任者，基本上没有官员被这些人皮吓倒而完全清廉。朱元璋曾做个梦，梦到800个没有头的人向他下跪。几天后，有来自全国的800多个没有完成征粮任务的粮官来见他，他毫不客气地下令全部处死。腐败结束了没有？没有。
再说腐败是人性的问题。中共在早期很清廉吧？也没有完全杜绝。刚建国时，毛泽东听说有官员用公款为自己建了房子，下令要枪毙，被周恩来拦住。这个人到底是谁？一直不清楚，后来看到有资料说，就是杨尚昆（这个不能得到证实，我也不知道看的资料是不是真的）。刘青山、张子善二人被枪毙，救了2万个甚至20万个官员，如果这个刚建国就用公款为自己盖房的人被枪毙，那么刘青山和张子善恐怕根本不敢贪污。
 
中国为治理公款吃喝，从1980年到现在，有多少文件？但从未真正杜绝。一阵风式的“运动”很容易，从一年搞几天的学雷锋，到建卫生城市，再到房地产调控，但长年杜绝腐败，我认为是不可能的。至于把茅台和腐败联系在一起，我认为是没有充足依据的。企业主请官员喝瓶茅台式的腐败，那是80年代式的腐败。现在还这么搞，恐怕被官员笑死、骂死。一瓶不到2000元的酒，就妄想从官员手中得到被法律和政策禁止的东西，这种人居然还能在中国做生意，智商未免太低了一些。如果把中国的腐败的焦点停留在茅台上，那我看中国反腐就基本上没有希望了。腐败不是喝了一点酒的问题，而是把本不应拿不到的批文（涉及10亿元资金）批出去，把依法应当征收的1亿元税款免除或减半，是把本应判死刑的人判3年，是依法应当取缔的企业继续运作10年，每年盈利1000万，是本应10亿元卖出的土地，以3亿元卖给房企。这还只是中级腐败。什么叫行贿？1000万的银行汇款叫行贿，1000瓶茅台送给官员，那叫小品，不叫行贿。
 
吃饭时喝酒，是内向的中国人与陌生人比较迅速地建立一定的信任基础的途径。这是人性。这里想起一个故事：
抗日战争时，彭德环到386旅检查工作。彭德怀这个人是个比较纯粹的人，脾气极不好，基本上没有人敢请他大吃大喝，更不用说以权谋私了。但是陈赓敢。彭德怀一到驻地就嚷着要见陈赓。结果被告知，陈赓临时有事外出了。一晃到了吃饭时间，彭德怀来到饭厅里就餐，一看，满桌都是粗茶淡饭，他感到很满意。
　　不一会，陈赓的警卫员悄悄贴着彭德怀说道：“陈司令回来了。”
　　“在哪里？”彭德怀睁大眼睛四处寻找。
　　警卫员用手一指饭厅里一个关着门的套间：“他在里边。”
　　“你把他叫出来。”彭德怀说道。
　　可是陈赓磨磨蹭蹭就是不出来。彭德怀起了狐疑，便推门进去，这时只见陈赓坐在桌子边津津有味地吃着鸡，彭德怀大怒地说道：“好呀陈赓，你这狗日的，你给我吃白水煮萝卜，自己却关起门来吃好的。你不让我吃我偏要吃，快给我拿一副碗筷来。”
　　一桌子好菜吃到一半，彭德怀突然停下筷子骂道：“陈赓，你这王八蛋，我像是又中了你的圈套。”
听到这话，陈赓笑得把饭都喷了出来。事后，陈赓逢人便得意洋洋地说道：谁说彭老总的清规戒律不能破？我不是破了吗？
 
说中央高官去地方视察，不喝茅台，我当然相信。但对于一般官员，一生之中再也不喝茅台，我不大相信。对于官员之间的交往，茅台也不大可能从此绝迹。如市委书记来视察，县委书记拿出茅台接待，顶多被批评两句，但如果不拿出来，就要冒着上级连批评都懒得批评的风险。哪个风险更大？而前一种情况，县委书记说两句“下不为例”，我看问题就不大。
 
关于茅台，最近听到一个说法，感觉比较有道理。为什么国外的高档葡萄酒始终在国内不能打开市场？除了牌子比较杂以外，中国的商务交往的要求也决定了茅台等高档白酒更有市场。比如两个公司要合作，在时间只有近2小时的饭局上打开局面，只有高度的白酒，而不是13度的葡萄酒。
葡萄酒不是更有营养吗？这基本上就是质疑茅台的人的扯淡。谁喝酒是为了补充营养？喝酒的人，有几个营养不良？到处是营养过剩需要减肥的人。谁不知道喝酒有害健康？看看身边的人，有多少人成功戒烟或是戒酒的？
 
茅台值得投资的最大利基是什么？是中国经济的高增长。中国人均收入每年增长10%的话，茅台的价格天然就应该上涨10%/年。
 
综上，最近茅台虽然逆市下跌，但我持续买进。当然，我对准确入市的时点的判断力，还是差了一些。要不然就不会提前2年介入银行股了。好在以后我基本上不用靠对市场的准确判断，也能在市场中持续赚钱了。按我的投资策略，股市4个月左右的时间内上涨10%，我投资的股票平均上涨16%以上的概率在80%以上。
