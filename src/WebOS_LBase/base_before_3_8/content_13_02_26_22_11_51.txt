 以下各职位主要工作地址为北京。 


招聘人数：1Web开发工程师 / PHP工程师
职位描述1. 根据产品需求，完成功能开发
 2. 数据库和服务器的日常维护 
职位要求1. 熟练使用CSS，JS，PHP，MYSQL
 2. 了解 MVC 结构框架，了解 LINUX 服务器
 3. 喜欢 WEB 开发，眼高不手低 


招聘人数：1产品经理
职位描述1. 负责译言产品的整体规划 
2. 了解社区运作, 能够进行用户需求调研和用户使用行为分析 
3. 负责产品定位、规划和功能设计
 4. 协调工程师和设计师，管理项目开发进度
 
职位要求1. 一年以上社区类产品相关经验 
2. 强大的逻辑思维和沟通能力 


招聘人数：1网页设计师
职位描述1.参与产品前期UE研究分析；
2.负责互联网产品UI的设计和规范
 3.可能涉及部分前端
职位要求1.有1~2年互联网产品UI设计经验；
 2.熟悉互联网社区，有良好的美术功底，有创新意识；
 3.熟练使用Photoshop、Illustrator等软件；
 4.会div/css优先



招聘人数：1精选编辑
职位描述：1、你将会从译言网社区搜罗各种有趣内容，把它们放到译言精选上
2、没错，译言的烂译文也有不少，你没准要和它们好好聊聊
3、你还要考虑用什么样的内容让译言精选和译言网变得更有意思
 职位要求：1、你要至少擅长一门外语
2、你要知道怎么找到用这门外语传播的好东西
3、你还要知道怎么把这些好东西介绍给更多不了解这门外语的人
4、你不一定是个“知识分子”，但至少要是个“知道分子”
5、如果你还是一个在某些方面有见解的“知道分子”，我们会加分
 

招聘人数：1媒介经理(Media Manager)
职位描述1. 制定译言网和东西网的社交媒体战略，并主导社交媒体日常维护；
2. 执行和运营市场品牌的相关项目。
职位要求1. 熟悉并认可译言网和东西网的气质；
2. 求知欲强，对世界充满好奇心，热爱数字媒体；
3. 具备一定的英文交流能力；
4. 最好具备很强的文字功底；


招聘人数：1广告总监
职位描述1、全面负责网站和线下活动的广告业务开发和管理；建立规范化的广告销售体系和客户管理体系；
2、负责参与重大广告合同的谈判和签订，确保并不断提高广告收益； 
3、及时跟进客户信息，制定个性化营销策略和解决方案； 
4、负责广告团队的建设。
职位要求1、本科及以上学历；
2、2年以上的广告销售经验，熟悉互联网广告行业；
3、拥有敏锐的市场触觉，强大的整合规划能力、市场分析能力和应变能力；
4、具有较强的业务谈判能力、广告企划能力、重要客户销售管理能力。



招聘人数：1图书编辑
职位描述 1. 你将负责东西文库、译言古登堡计划的图书选题策划、译文审读或图书营销推广等工作
职位要求1. 热爱新媒体和电子出版
2. 中、英文俱佳，有翻译和编辑经验，能胜任译文审校工作
3. 涉猎广泛，思维活跃，能独立完成图书选题的策划和推广
4. 在人文或科技任一方面有所擅长
5. 懂技术或设计可加分

 你可以浏览东西文库的 [广播] 和译言古登堡计划的 [微博] 进一步了解我们的日常工作，期待你的加入！
